Checkbox (or radio, via `radio`) with accent fill and an animated check/dot.

```jsx
<Checkbox label="Remember this device" defaultChecked />
<Checkbox radio name="plan" value="pro" label="Pro" />
<Checkbox radio name="plan" value="team" label="Team" defaultChecked />
```

Set `radio` for round radio behavior (group with shared `name`). Controlled or uncontrolled like a native input.
