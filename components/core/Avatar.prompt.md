User or entity avatar with image, initials fallback, and optional presence dot.

```jsx
<Avatar name="Ada Lovelace" status="online" />
<Avatar src="/u/42.jpg" name="Ada" size="lg" />
<Avatar name="OM" accent size="sm" />
```

Sizes: `sm` 28 / `md` 36 / `lg` 48 / `xl` 64. `status`: online / busy / away / offline. `accent` fills the initials state with the brand gradient.
