Icon-only square button for toolbars, headers, and dense controls.

```jsx
<IconButton icon="settings" label="Settings" />
<IconButton icon="plus" variant="accent" label="Add" />
<IconButton icon="x" variant="solid" size="sm" label="Close" />
```

Variants: `ghost` (default), `solid` (glass), `accent`. Sizes: `sm` 32 / `md` 40 / `lg` 48. Pass a Lucide name string or any React node as `icon`. Always set `label`.
